== Description ==
As the name explains, <strong>Article Read Time Lite</strong> is a free WordPress plugin which calculates the estimated reading time required to read the article in your site 
and presents them in a beautiful manner with our available <strong>Paragraph</strong> and <strong>Block</strong> Templates. Currently there are all together 4 Beautiful Pre Designed 
templates from which you can choose any as per your desire.


= See Article Read Time Lite features list below: =
* 2 Pre Designed Paragraph Templates
* 2 Pre Designed Block Templates
* Enable/Disable Article Read Time in Posts/Page or any other post types
* Enable/Disable Progress Bar in Posts/Pages or any other Post Types
* Display Word, Characters and Reading Time
* Display Article Read Time before content and after content
* Customize Text as per your need
* Layout customization options available for Progress Bar
* All device friendly and browser Compatibility
* Mobile Responsive
* Translation Ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/article-read-time-lite` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Article Read Time Lite settings page to configure the plugin



== Frequently Asked Questions ==

= Where I can display the article read time in frontend ? =
You can choose the option to display above the content, below the content or in both section.

= How is the reading time calculated? =
You can configure the word per minute in backend with which our plugin calculates as per the words available in the post content.

= Can I enable progress bar?
Yes our plugin has the progress bar option too.

= Do I need to design the article read time?
Our plugin already have 4 pre designed templates from which you can choose one.
